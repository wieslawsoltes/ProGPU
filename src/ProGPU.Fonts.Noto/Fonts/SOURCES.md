# Bundled Noto font sources

- `NotoSansCJKjp-Regular.otf`: unmodified Noto Sans CJK JP 2.004 Regular from the official
  [notofonts/noto-cjk](https://github.com/notofonts/noto-cjk) project.
  SHA-256: `68a3fc98800b2a27b371f2fb79991daf3633bd89309d4ffaa6946fd587f375b5`.
- `NotoSansSymbols2-Regular.ttf`: unmodified full TrueType Noto Sans Symbols 2 2.006 Regular
  from the official [notofonts/symbols](https://github.com/notofonts/symbols) release.
  SHA-256: `2af28573fcdb6c72ec195908c2f95a5d82fb65c8e289a11aa7088663f4cab99b`.
